# Description 

Basic festival api with Node + Express + Mongoose

# Methods : 

GET, POST, PUT, DELETE

# Routes: 

- post
- getAlls
- search (option: ?term=)
- getById/:id
- updateById/:id
- deleteById/:id


# COMMANDS


Install the node modules 

```console
    npm install
```

Then run the project with it

```console
    npm run dev
```

# Entity

(Form urlencoded)

Festival :

name: String,
startDate: Date,
endDate: Date,
numberPlaces: Integer
