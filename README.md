# Description 

Basic festival api with Node + Express + Sequelize

# Methods : 

GET, POST, PUT, DELETE

# Routes: 

- post
- getAlls
- getById/:id
- updateById/:id
- deleteById/:id


# COMMANDS


Install the node modules 

```console
    npm install
```

At first run this command for migrate all entities on database.

```console
    npm run migrate
```

Then run the project with it

```console
    npm run dev
```

# Entity

Festival :

name: String,
startDate: Date,
endDate: Date,
numberPlaces: Integer
